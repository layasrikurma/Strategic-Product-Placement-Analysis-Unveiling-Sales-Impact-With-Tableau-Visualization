Project Overview

This project analyzes how product placement, promotions, and consumer demographics influence sales performance using Tableau visualizations. The insights help businesses optimize product positioning strategies to improve revenue.
Objectives

Analyze sales impact based on product position

Study promotion vs sales performance

Understand consumer demographic behavior

Deliver interactive dashboards & story

Embed Tableau visuals using Flask UI
